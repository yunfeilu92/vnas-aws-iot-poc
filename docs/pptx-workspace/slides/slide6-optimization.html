Cost Optimization Recommendations

Compress Messages to <5KB

Reduce IoT Core billing blocks from 26 to 1. Use Protocol Buffers or gzip at vehicle edge.

IoT Core: $28K to $1.1K (-96%)

Batch Aggregation (10s)

Aggregate 10 messages into 1 batch before publishing. Reduces message count by 10x.

Messages: 1.08B to 108M

Off-Peak Scheduling

Stagger vehicle data transmission to reduce peak throughput. Smaller MSK instances.

MSK: smaller broker type

S3 Parquet Format

Columnar storage with ~50% compression. Reduces storage cost and Athena scan volume.

Storage: 129TB to ~64TB

Athena Partition Strategy

Partition by date and vehicle ID. Use projection for dynamic partition discovery.

Query cost: up to -90%

Combined Optimization

Compression + batching can reduce total monthly cost from ~$44K to under $15K.

Total: ~$44K to ~$15K (-66%)

Key Insight: Message compression alone saves ~$27K/month. Prioritize vehicle-edge optimizations first.
