关键技术点

Fleet Indexing

索引 Shadow 和 Registry 属性，支持基于 firmwareVersion 的动态查询。延迟仅几秒钟。

设备自动分组基础

Dynamic Thing Group

基于查询条件自动匹配设备。Shadow 属性变化时自动加入/离开，无需手动管理。

零人工干预设备管理

Presigned URL

临时 S3 访问授权，可设置过期时间（推荐 2-24 小时）。设备无需配置 AWS 凭证。

安全的固件分发

MQTT Jobs Topics

notify-next 被动接收 + $next/get 主动查询。双模式确保设备不漏掉任何 Job。

100% Job 送达率

Device Shadow

设备状态持久化存储。升级后更新 firmwareVersion，Fleet Index 自动反映最新状态。

设备状态单一来源

版本去重

设备重启后未完成 Job 会重新下发。通过检查 currentVersion 防止重复下载安装。

幂等性保障

核心模式：Shadow + Fleet Index + Dynamic Group = 全自动设备生命周期管理
