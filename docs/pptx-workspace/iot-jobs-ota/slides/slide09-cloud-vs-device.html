云端配置 vs 设备端流程

Cloud Side (管理员操作)

1. 启用 Fleet Indexing

索引 Device Shadow 的 firmwareVersion 字段

aws iot update-indexing-configuration

2. 上传固件到 S3

创建 S3 Bucket，上传固件二进制文件

aws s3 cp firmware.bin s3://bucket/

3. 创建 Dynamic Thing Group

查询：shadow.reported.firmwareVersion:1.0.2

aws iot create-dynamic-thing-group

4. 创建 IoT Job

指定 target Group + Job Document (Presigned URL + checksum)

aws iot create-job --targets ...

Device Side (设备自动执行)

1. MQTT 连接 + 订阅

X.509 证书认证，订阅 jobs/notify-next 和 $next/get/accepted

2. 接收 Job 通知

被动推送 (notify-next) 或主动查询 ($next/get)

3. 下载 + 校验固件

通过 Presigned URL 下载，SHA256 校验完整性

4. 安装 + 重启

写入新固件、更新启动脚本、触发设备重启

5. 上报结果 + 更新 Shadow

上报 SUCCEEDED，更新 firmwareVersion，自动离开旧 Group
